# DSA Instructor AI 🤖

A full-stack AI-powered DSA Instructor Chatbot built using:

- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- Gemini API

The chatbot answers only Data Structures & Algorithms related questions in a beginner-friendly way.

---

# Features

✅ Beautiful modern UI  
✅ AI-powered DSA explanations  
✅ Beginner friendly answers  
✅ Optimized approach explanations  
✅ Code generation  
✅ Dry runs  
✅ Secure backend API handling  
✅ Full-stack deployment on Render
